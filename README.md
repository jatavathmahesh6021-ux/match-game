# 🎮 React Match Game

A fast-paced, interactive matching game built with React.js! Test your memory and reflexes by matching the target image with the correct thumbnail before the 60-second timer runs out. 

### 👉 [Play the Live Game Here!](https://jatavathmahesh6021-ux.github.io/match-game/)

## ✨ Features
* **Dynamic Timer:** A 60-second countdown timer that automatically ends the game when it reaches zero.
* **Complex State Management:** Accurately tracks the user's live score and manages win/loss conditions.
* **Categorized Tabs:** Filter images instantly by categories like Fruits, Animals, and Places using React state.
* **Randomized Targets:** Generates a new random target image from the data array upon every successful match.
* **Responsive Design:** Fully styled with custom CSS for a seamless experience on both desktop and mobile.

## 🛠️ Tech Stack
* **Frontend:** React.js, JavaScript (ES6+), HTML5, CSS3
* **Tools:** Node.js, Git
* **Deployment:** GitHub Pages
